文件动作

上传文件 upload_file

动作请求：

字段类型说明
typeString上传方式：url | path | data
nameString文件名称
urlString可选，上传方式为 url 时需要提供的 url
dataMap<String, String>可选，url 可选 headers
pathString可选，上传方式为 path 时需要提供的路径
dataString可选，上传方式为 data 时需要提供的数据
sha256String可选，文件的 sha256 值
file_typeString可选，文件类型，缺省值为 image

动作响应：

字段类型说明
file_idString文件 ID
